# patika.dev-merge-sort

[16 ,21 ,11 ,8 ,12 ,22] dizisinin merge sort yöntemiyle sıralanması.

sub-array oluşturabiliriz;
[16, 21, 11] ve [8, 12, 22]

[16, 21] [11]  [8] [12, 22]

[16] [21]  [11]    [8]  [12] [22]

[16, 21] [11]   [8]  [12, 22]

[11, 16, 21]   [8, 12, 22]

[8, 11, 12, 16, 21, 22]

Big O Notation: O(nlogn)
